# E-dit Mobile

E-dit — это мощный, но минималистичный инструмент для быстрого форматирования, обработки и редактирования текста на мобильных устройствах. Приложение построено с использованием современных веб-технологий (React, TailwindCSS, Vite) и упаковано в нативное Android-приложение с помощью **Capacitor**.

## Основные возможности

- **Быстрые макросы и команды**: Мгновенная очистка текста от лишних пробелов, скобок, знаков препинания, приведение к нужному регистру.
- **Инструменты для Suno AI**: Специализированные команды для обработки текстов песен (очистка тегов, отбивка куплетов пустой строкой, перевод тегов в CAPS).
- **Анализатор спецсимволов**: Умная панель, которая находит в тексте нестандартные символы и позволяет в один клик временно скрыть их из текста для удобства чтения.
- **Кастомная история изменений**: Продвинутые Undo/Redo с точным сохранением позиции курсора.
- **Высокая производительность**: Нативный мобильный опыт благодаря отказу от тяжелых CSS-эффектов и правильному использованию DOM-элементов.

## Технологический стек

- **Frontend Framework**: React 18
- **Сборщик**: Vite
- **Стилизация**: Tailwind CSS (без использования компонентов, чистые utility classes)
- **Иконки**: Lucide React
- **Мобильная обертка**: Capacitor (Android)

## Как запустить локально

### 1. Веб-версия (Браузер)
```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev
```

### 2. Сборка Android APK
Для сборки APK-файла у вас должен быть установлен Java JDK (например, JDK 17) и Android SDK.

```bash
# Сборка продакшен бандла React
npm run build

# Синхронизация файлов с Capacitor проектом
npx cap sync

# Сборка APK с помощью Gradle (убедитесь, что переменная среды JAVA_HOME задана верно)
cd android
./gradlew assembleDebug
```
Готовый `.apk` файл появится по пути: `android/app/build/outputs/apk/debug/app-debug.apk`

## Структура проекта

- `src/components/` — UI компоненты (Кнопки команд, Тулбар, Панель символов, Модальные окна)
- `src/lib/commands/` — Логика функций по обработке текста (регулярные выражения и строковые методы)
- `src/hooks/` — Кастомные React хуки (`useEditor.ts` для истории изменений)
- `src/styles/` — Глобальные стили Tailwind
- `android/` — Нативный Android-проект Capacitor

Полное описание каждой функции приложения можно найти в файле [FUNCTIONS.md](./FUNCTIONS.md).
